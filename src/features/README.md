# Features

Основные фичи/секции приложения свадебного приглашения.

## Секции

### Hero
Главный экран с именами молодоженов, датой свадьбы и призывами к действию.

**Файл**: `hero/Hero.tsx`

### Countdown
Живой таймер обратного отсчета до дня свадьбы. Обновляется каждую секунду.

**Файлы**:
- `countdown/Countdown.tsx`
- Hook: `hooks/useCountdown.ts`

### Details
Информация о времени и месте проведения свадьбы с возможностью открыть на карте.

**Файл**: `details/Details.tsx`

### Gallery
Фото-галерея с возможностью просмотра в полноэкранном режиме.

**Файл**: `gallery/Gallery.tsx`

## Добавление новой секции

1. Создайте директорию в `features/`
2. Создайте основной компонент секции
3. Импортируйте в `app/page.tsx`
4. При необходимости добавьте в навигацию в Header

## Принципы

- Каждая фича самодостаточна
- Использует общие компоненты из `components/ui/`
- Получает данные из `constants/`
- Типы определены в `types/`
